## Aim
The aim of this project is to simulate banking interest calculation, allowing the user to compute the final balance by adding yearly interest to the principal amount over a specified number of years.

## How the Program Works
1. User enters principal amount, rate of interest, and number of years.
2. Program stores the principal as the initial balance.
3. A loop runs for each year.
4. Interest is calculated yearly.
5. Interest is added to the balance.
6. Final balance is displayed.

## Expected Outcome
The program displays the final balance after applying yearly interest.
